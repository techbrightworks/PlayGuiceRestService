PlayGuiceRestService
-----------------------------
1.Unzip the Zip file
2.From the command Line move to PlayGuiceRestService directory and build the Project.
3.Build Command: sbt compile , Eclipse -> sbt compile eclipse
4.Deploy and Run the application Using sbt run
5.Check the REST API with POSTMAN REST Client 
Example1:http://localhost:9000/change/optimal?pence=576
Example2:http://localhost:9000/change/supply?pence=576
6.Source: Java8
